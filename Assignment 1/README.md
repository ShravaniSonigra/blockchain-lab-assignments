## Assignment 1: Basic Smart Contract

## Objective
To create a smart contract that stores and retrieves data on the blockchain.

## Tools Used
Solidity  
Remix IDE  
MetaMask  

## Process

## Create Contract
Created a Solidity contract named Storage  
Defined a state variable number  

## Store Data
Used store(uint256 num) function  
Stores value in blockchain  

## Retrieve Data
Used retrieve() function  
Returns stored value  

## Smart Contract Functions
store(uint256 num) → stores value  
retrieve() → returns value
